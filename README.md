# Nepal-Economics-Insight
 Nepal Economic Insights Dashboard

📌 Project Overview

The Nepal Economic Insights Dashboard is a data-driven web application designed to analyze, visualize, and provide insights into Nepal’s economic indicators. The goal is to help users understand trends in GDP, inflation, trade, employment, remittance, and other macroeconomic variables through interactive visualizations.

This project demonstrates skills in data analysis, data visualization, and dashboard development using real-world datasets.

⸻

🎯 Objectives

* Analyze Nepal’s economic performance over time
* Build interactive and user-friendly dashboards
* Extract meaningful insights from raw data
* Showcase data analytics and visualization skills for academic and professional purposes

⸻

📊 Key Features

* 📈 GDP Growth Analysis (yearly trends)
* 💸 Inflation Rate Tracking
* 🌍 Import vs Export comparison
* 💼 Employment/Unemployment trends
* 💰 Remittance inflow analysis
* 📉 Data filtering (year-wise, sector-wise)
* 📊 Interactive charts (line, bar, pie, heatmap)

⸻

🧰 Tech Stack

* Python (Pandas, NumPy)
* Data Visualization: Plotly / Matplotlib / Seaborn
* Dashboard Framework: Streamlit / Dash
* Data Source: CSV / API
* Version Control: Git & GitHub
